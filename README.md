# A Day in LA Tours — Email Signature Assets

Hosted images for the customer-service Gmail email signature.

These are served at full resolution via GitHub Pages so Gmail does not
re-compress / pixelate them (the previous version embedded the images as
base64, which Gmail downsamples on paste).

| File | Used for | Display size | Source size |
|------|----------|-------------|-------------|
| `sig-logo.png` | A Day in LA Tours logo | 160px wide | 1220×693 |
| `sig-facebook.png` | Facebook icon | 17×17 | 96×96 |
| `sig-instagram.png` | Instagram icon | 17×17 | 96×96 |
| `sig-whatsapp.png` | WhatsApp icon | 17×17 | 96×96 |
| `sig-google.png` | Google reviews icon | 14×14 | 56×56 |
| `sig-tripadvisor.png` | Tripadvisor icon | 14×14 | 128×128 |
| `sig-getyourguide.png` | GetYourGuide icon | 14×14 | 56×56 |
